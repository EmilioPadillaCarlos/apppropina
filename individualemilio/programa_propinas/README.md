# Tip & Split Emilio Padilla Carlos

App Flutter de una sola pagina para calcular propina y dividir la cuenta.

Funciones principales:
- Campo para ingresar el total de la cuenta (solo numeros).
- Slider para elegir porcentaje de propina de 0% a 30%.
- Control de personas con botones de "+" y "-".
- Resultados con propina total, total general y pago por persona.
